## Run the tests

1. Read and follow the [prerequisites](../../#how-to-run-the-tests).

1. Install dependencies:

        yarn

1. Run the tests:

        yarn test

1. Start the damn thing

        yarn start

1. Deploy the fucking thing

        yarn deploy

1. Undeploy the fucking thing

        yarn undeploy
